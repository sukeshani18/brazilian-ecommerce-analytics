# Brazilian E-Commerce Analytics Dashboard
![Dashboard](Dashboard_Showcase.png)

## Project Overview

An end-to-end Data Analytics project built using Python, PostgreSQL, and Power BI to analyze Brazilian e-commerce transactions from 2016–2018.

## Objectives

* Analyze sales performance
* Evaluate profitability trends
* Understand customer behavior
* Identify top-performing states, cities, and product categories

## Tools & Technologies

* Python (Pandas)
* PostgreSQL
* Power BI
* DAX

## Dataset

Brazilian E-Commerce Dataset (2016–2018)

## Key Metrics

* Total Sales: 16.65M
* Estimated Profit: 14.53M
* Total Orders: 99.44K
* Profit Margin: 87.3%

## Dashboard Pages

### 1. Executive Summary

* Sales KPIs
* Sales Trend Analysis
* Top States by Sales
* Product Category Analysis
* Payment Method Distribution

### 2. Profit Analysis

* Estimated Profit
* Profit Margin
* Top Profit State
* Top Profit Category
* Profit Trend Analysis

### 3. Customer Analysis

* Unique Customers
* Top City
* Top State
* Customer Distribution Analysis

## Business Insights

* São Paulo (SP) generated the highest sales and profit.
* Credit Card payments accounted for the majority of transactions.
* beleza_saude was the most profitable product category.
* Customer concentration was highest in São Paulo.
